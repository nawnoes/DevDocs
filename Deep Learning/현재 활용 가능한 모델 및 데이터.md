# NLP 관련 활용 가능한 모델 및 데이터
## 1. 모델
Transformer 기반의 BERT와 GPT-2 등을 우선적 조사 
  - DistilKoBert: 기존 12개 layer 였던 버트를 3개 레이어로 축소. 
  - koGPT-2: 여러가지 GPT-2 모델 사이즈 중 가장 작은 117M 모델 공개
